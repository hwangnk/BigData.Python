# Jupyter 주요 shortcut

SHIFT + ENTER
현재 셀을 실행 또는 마크다운을 컴파일 한다.

CTRL + s
문서 저장.

CTRL + z
방금 한 것 취소.

ESC + a
위로 셀 삽입.

ESC + b
아래로 셀 삽입.

ESC + d + d
현재의 셀 삭제.

ESC + m
현재의 셀을 마크다운 셀로 전환.

ESC + y
현재의 셀을 코드 셀로 전환.

•ESC를 누르면 셀의 밴드가 파란색으로 변하고 쇼트컷을 연속적으로 입력할 수 있다.
•커서로 셀을 가리키며 클릭하면 셀의 밴드가 녹색으로 변하고 편집 할 수 있는 상태가 된다.
